Specifically, the figures and tables in our paper and supplementary material can be reproduced using the following scripts:
- Running `fig2.R` reproduces Figure 2. The datasets (GALAXY_mOTUs_v25.tsv, GALAXY_load.tsv) used to generate Figure 2 and construct the AA templates can be downloaded from Zenodo Record 14280080 (https://zenodo.org/records/14280080). 
- Running `global_fig_and_table.R` reproduces Figure 4, Figure S2, and Table S2,S3.
- Running `taxon_fig_and_table.R` reproduces Figure 3, Figure 5, Figure S1,S3,S4, and Table S3.
- Running `realdata_fig_and_table.R` reproduces Figure 6 and Table S4. Note that this script requires the output generated by `data_analysis.R` in the `real_data_analysis/` subfolder. Therefore, `data_analysis.R` must be executed prior to running `realdata_fig_and_table.R`.
