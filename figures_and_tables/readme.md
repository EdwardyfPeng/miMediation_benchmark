Specifically, the figures and tables in our paper and supplementary material can be reproduced using the following scripts:
- Running `fig1.R` reproduces Figure 1. The datasets (GALAXY_mOTUs_v25.tsv, GALAXY_load.tsv) used to generate Figure 1 and construct the AA templates can be downloaded from Zenodo Record 14280080 (https://zenodo.org/records/14280080). 
- Running `global_fig_and_table.R` reproduces Figure 3, Figure S3, and Table S2,S3.
- Running `taxon_fig_and_table.R` reproduces Figure 2, Figure 4, Figure S2,S4,S5, and Table S3.
- Running `realdata_fig_and_table.R` reproduces Figure 5 and Table S4. Note that this script requires the output generated by `data_analysis.R` in the `real_data_analysis/` subfolder. Therefore, `data_analysis.R` must be executed prior to running `realdata_fig_and_table.R`.
